# 📝 TaskFlow - Modern Task Manager & To-Do App

**TaskFlow** คือแอปพลิเคชันจัดการรายการสิ่งที่ต้องทำ (To-Do / Task Manager) ยุคใหม่ ที่เน้นการออกแบบสไตล์ Modern Minimalist มีความสวยงาม ใช้งานง่าย และช่วยเพิ่มประสิทธิภาพในการทำงาน (Productivity) ของคุณได้อย่างลงตัว ด้วยหน้าตา UI ที่เรียบหรู พร้อมฟังก์ชันจัดกลุ่มงาน ค้นหา และวิเคราะห์ข้อมูลส่วนตัว

---

## ✨ คุณสมบัติเด่น (Features)

- 📥 **การจัดการข้อมูลงานระดับพรีเมียม (Advanced Task Management):** เพิ่ม, แก้ไข, ลบ และทำเครื่องหมายว่าเสร็จสิ้นงานได้อย่างราบรื่น
- 🏷️ **หมวดหมู่และป้ายกำกับ (Categories & Tags):** จัดกลุ่มงานตามต้องการ เช่น Work, Personal, Shopping, Health
- ⚡ **จัดลำดับความสำคัญ (Priority Levels):** แบ่งระดับความสำคัญของงาน (สูง, กลาง, ต่ำ) พร้อมสีสันนำสายตาที่สวยงาม
- 🔍 **ระบบค้นหาและตัวกรอง (Search & Filter):** ค้นหางานด้วยคีย์เวิร์ด หรือกรองงานตามสถานะ หมวดหมู่ และลำดับความสำคัญได้อย่างรวดเร็ว
- 📊 **แดชบอร์ดสถิติ (Statistics Dashboard):** แสดงกราฟและเปอร์เซ็นต์ความสำเร็จของงาน เพื่อช่วยกระตุ้นความโปรดักทีฟของคุณ
- 💾 **บันทึกข้อมูลอัตโนมัติ (Data Persistence):** บันทึกข้อมูลงานลงใน LocalStorage ของเบราว์เซอร์โดยอัตโนมัติ ไม่ต้องกังวลว่าข้อมูลจะหายเมื่อรีเฟรชหน้าเว็บ
- 🎨 **การออกแบบที่สวยงาม (Premium UI/UX):**
  - รองรับโหมดมืด (Dark Mode) และโหมดสว่าง (Light Mode)
  - เอฟเฟกต์ Glassmorphism แบบกระจกฝ้าที่ดูพรีเมียม
  - อนิเมชันที่ลื่นไหล ตอบสนองทุกสัมผัส (Micro-interactions)
  - แสดงผลได้ดีเยี่ยมบนมือถือ แท็บเล็ต และคอมพิวเตอร์ (Responsive Design)

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

เราขอแนะนำเทคโนโลยีต่อไปนี้สำหรับการพัฒนา (สามารถปรับเปลี่ยนได้ตามความต้องการของคุณ):

- **Frontend:** Vanilla HTML5, CSS3, และ Modern JavaScript (ES6+) หรือ [React.js](https://react.dev/) / [Next.js](https://nextjs.org/)
- **Styling:** Vanilla CSS (สำหรับความยืดหยุ่นสูงสุดและการสร้างอนิเมชันเฉพาะตัว) หรือ TailwindCSS
- **Icons:** Google Material Icons หรือ Font Awesome
- **Development Tool:** [Vite](https://vitejs.dev/) (สำหรับเครื่องมือสร้างโปรเจกต์ที่รวดเร็ว)

## 🚀 เริ่มต้นใช้งาน (Getting Started)

### การเตรียมความพร้อมก่อนการติดตั้ง (Prerequisites)
- [Node.js](https://nodejs.org/) (แนะนำเวอร์ชัน LTS)
- Web Browser ทั่วไป (Chrome, Safari, Edge, Firefox)

### วิธีการรันโปรเจกต์ (แบบ Vanilla CSS / JS ด้วย Vite)

1. **โคลนและดาวน์โหลดโปรเจกต์**
   ```bash
   git clone https://github.com/yourusername/Task-Flow.git
   cd Task-Flow
   ```

2. **ติดตั้ง Dependencies**
   ```bash
   npm install
   ```

3. **รันเซิร์ฟเวอร์จำลองการพัฒนา (Development Server)**
   ```bash
   npm run dev
   ```

4. **เปิดเบราว์เซอร์**
   เข้าไปที่ `http://localhost:5173` (หรือตามพอร์ตที่ระบุใน Terminal) เพื่อเริ่มต้นทดสอบแอปพลิเคชัน

## 📂 โครงสร้างโฟลเดอร์ (Folder Structure)

```text
To-Do-App/
├── index.html          # ไฟล์หลักแสดงผลหน้าเว็บ
├── README.md           # รายละเอียดโปรเจกต์ (ไฟล์นี้)
├── package.json        # การตั้งค่า Dependencies ของ Node.js
└── src/
    ├── css/
    │   └── main.css    # ไฟล์สไตล์หลักและการตั้งค่าระบบสี/การออกแบบ
    ├── js/
    │   ├── app.js      # โลจิกควบคุมหน้าเว็บและการจัดการเหตุการณ์ (Event Handling)
    │   └── storage.js  # โมดูลสำหรับจัดการข้อมูลกับ LocalStorage
    └── assets/         # รูปภาพ ไอคอน และไฟล์มีเดียต่างๆ
```
